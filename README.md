# Xamarin F50

A cross platform 'batteries included' library for consuming commonly used platform features and API's in shared code.

## Supported Platforms
 - iOS (10+)
 - Android (4.4+)
 - UWP (Fall Creators Update+)

## Current Features:
 - Permissions
 - Preferences
 - Geolocation
 - Connectivity
 - Device Info
 - SMS / Phone / Email
 - Sharing
 - Text-to-Speech

## Planned Features:
 - Media Picker
 - Browser
